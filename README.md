# zedstore
thinkphp5+amazeUi做的技能服务商城
适合新手学习tp5
你可以在该商场获得游戏陪玩，线下陪玩，家教服务，跑腿代办，摄影约拍等私人服务

环境要求：<br>
nginx+mysql

### 关于百度地图api
门店地址调用了百度地图api接口，需要自己申请密钥
然后搜索下面这行，取消注释，填写你的密钥
```
<script type="text/javascript" src="https://api.map.baidu.com/api?v=2.0&ak=你的密钥"></script>
```
